## Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/guilhermecapitao/nlw04-moveit-web.git && cd moveit-next
```

Follow the steps below
```bash
# Install the dependencies
$ yarn

# Start the project
$ yarn dev
```
